🎮 Perfil Games
O Perfil Games é um projeto criado com o objetivo de ajudar jogadores a descobrirem seu perfil gamer através de um quiz interativo. No site, o usuário responde algumas perguntas sobre seus gostos em jogos e, no final, recebe uma recomendação de jogos que combinam com seu estilo.
Escolhi esse tema porque jogos fazem parte do meu cotidiano desde muito novo, então achei interessante desenvolver um projeto com algo que eu gosto e tenho familiaridade.
O projeto também possui sistema de cadastro e login de usuários, armazenamento das respostas do quiz em banco de dados e uma dashboard mostrando informações e gráficos sobre os perfis gamers mais escolhidos.

🚀 Funcionalidades

- Cadastro de usuários

- Login

- Quiz interativo

- Descoberta do perfil gamer

- Recomendação de jogos

- Dashboard com gráfico de pizza

- Armazenamento de dados no banco
